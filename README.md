# AQR - Al Quran Reader

A beautiful, lightweight, and responsive web application to read, search, and bookmark the Holy Quran. Designed with a clean Green/White theme and custom SVG icons for a premium user experience.

## Features

- **Complete Quran:** Read all 114 Surahs with Arabic text and English translations.
- **Search:** Quickly find specific words, phrases, or topics within the Quran.
- **Bookmarks:** Save your favorite Ayahs locally for quick and easy access.
- **Clean UI/UX:** A calming Green/White theme designed for readability and focus.
- **Custom Icons:** Uses custom SVG icons throughout the interface.
- **Responsive Design:** Fully optimized for both desktop and mobile devices.
- **Share & Copy:** Easily share or copy Ayah references.

## Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **API:** Al Quran Cloud API (api.alquran.cloud)
- **Hosting:** GitHub Pages

## How to Run Locally

This project is a single-page application and requires no build steps or complex setup.

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Start reading!
